IntelliJ IDEA安装破解步骤

1、官网下载IntelliJ IDEA

下载地址 点击Download

注意：所有安装软件、破解软件在文末都有提供（会不定时更新到最新版本保证一直可用）。

2、选择对应的操作系统下载

3、运行安装包进行安装

点击Next下一步，选择安装路径，注：文件夹下不要存在中文，避免出现不必要的麻烦。

点击Next下一步，安装选项设置

  • Create Desktop Shortcut：创建桌面快捷方式，根据操作系统选择
  • Update context menu：鼠标右键添加“打开文件夹作为项目”选项，看个人需要；
  • Create Associations：创建关联，勾选后打开对应文件都是以IDEA打开，个人建议不勾选；
  • Download and install 32-bit JetBrains Runtime：下载并安装32位运行环境，安装JDK已经安装过就无需勾选。个人不建议勾选，此处安装是在JetBrains下载，国内网速普遍下载缓慢；
  • Update PATH variable(restart needed)：更新路径变量（需要重启），添加IDEA的环境变量。
  • 选择完成后点击next进行下一步，

然后无脑下一步就可以了。

基础设置：鼠标滑轮修改字体大小

设置坐标：File -> Settings -> Editor -> General

或通过Ctrl + Shift + A 输入框输入mouse 找到Change font size with…..

基础设置：鼠标悬浮提示(有待测试)

设置坐标：File -> Settings -> Editor -> Code Editing

基础设置：自动导包

设置坐标：File -> Settinngs -> Editor -> General -> Auto Import

第一项：表示IDEA将在我们写代码的时候自动帮我们导入需要的包。但是对于那些同名的包，还是需要手动Alt + Enter进行导入。

第二项：表示IDEA将在我们写代码的时候帮我们优化导入的包，比如去掉一些没有用的包（等同于自动 Ctrl + Alt + O ）。
