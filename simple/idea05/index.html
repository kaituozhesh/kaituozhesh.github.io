自定义类 | 接口模板

每个类和接口都应该有对应的描述信息、作者、版本等信息，对于这样的重复代码，这里通过IDEA提供给的代码模板来完成。

设置坐标：File -> Settings -> Editor -> File and Code Templates -> Files -> Class

类模板配置信息

保存后，当我们创建类的时候就会自动生成类注释了

内置属性

  • ${PACKAGE_NAME}：创建类的包名
  • ${NAME}：类名
  • ${USER}：当前系统的登录用户名
  • ${DATE}：当前系统日期
  • ${TIME}：当前系统时间
  • ${YEAR}：当前年
  • ${MONTH}：当前月
  • ${MONTH_NAME_SHORT}：当前月分英文缩写，如：Jan, Feb
  • ${MONTH_NAME_FULL}：当前月份英文全称，如：January, February
  • ${DAY}：本月的当前日期
  • ${HOUR}：当前小时
  • ${MINUTE}：当前分钟
  • ${PROJECT_NAME}：当前项目名称
