01.IntelliJ IDEA入门篇：IntelliJ IDEA安装及破解


阅读数: 次   
字数：475字 | 预计阅读时长：1分钟
2021-03-10

IntelliJ IDEA安装步骤

官网下载IntelliJ IDEA

下载地址 点击Download

选择对应的操作系统下载

运行安装包进行安装

点击Next下一步，选择安装路径，注：文件夹下不要存在中文，避免出现不必要的麻烦。

点击Next下一步，安装选项设置

  • Create Desktop Shortcut：创建桌面快捷方式，根据操作系统选择
  • Update context menu：鼠标右键添加“打开文件夹作为项目”选项，看个人需要；
  • Create Associations：创建关联，勾选后打开对应文件都是以IDEA打开，个人建议不勾选；
  • Download and install 32-bit JetBrains Runtime：下载并安装32位运行环境，安装JDK已经安装过就无需勾选。个人不建议勾选，此处安装是在JetBrains下载，国内网速普遍下载缓慢；
  • Update PATH variable(restart needed)：更新路径变量（需要重启），添加IDEA的环境变量。
  • 选择完成后点击next进行下一步，

然后无脑下一步就可以了。

IntelliJ IDEA破解步骤

下载破解工具包（包含了破解插件、激活码）

点击试用进入IDEA

下载好后运行软件，进入idea主界面，提示需要注册，这时候只需要勾选试用（Evaluate for free）选项，点进Evaluate进入IDEA。

注：如果不是第一次破解，请先执行工具包解压目录 -> 使用教程 -> reset_script 对应文件重置一下试用时间。

导入下载的插件包

坐标：File -> Settings -> Plugins -> Install Plugin form Disk

找到之前下载解压的工具包，选择安装即可

重启软件

激活

打开主界面，找到 Help->Register，选择activation code，将key复制到框内，点击activate即可，如图

key在工具包中有提供
