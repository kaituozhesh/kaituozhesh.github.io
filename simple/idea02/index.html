02.IntelliJ IDEA入门篇：项目创建


阅读数: 次

2021-03-10

创建普通Java项目

  • 首先还是一样 File -> New -> Project 进入到创建项目界面

  • 选择Java并且选择JDK安装路径，点击New添加，JDK选择好之后点击Next

  • 是否选择模板，这里不要勾选，直接Next

  • 输入项目名称和项目存储路径，点击Finish完成创建

创建Maven项目

  • 首先还是一样 File -> New -> Project 进入到创建项目界面

  • 选择Maven并且选择JDK，模板选择根据自己的需求，这里就不选择了。

  • 输入GroupId和ArtifactId

  • 选择项目名称和项目存放地址

  • Finish完成后就是一个空的maven模板（一般企业开发中都是创建空模板）

创建Spring Boot项目

注： 这里有两种方式，一种是直接创建SpringBoot项目并引入相关基础依赖（适用于单体）。一种是创建Maven项目，自己引入pom和创建配置文件及启动类（适合多模块分布式项目）。

直接创建Spring Boot项目

  • 万年不变的第一步File -> New -> Project

  • 选择Spring Initializr并且选择JDK版本

  • 设置项目基本信息

  • type：创建的项目类型（Maven/Gradle）

  • Language：使用的语言

  • packaging：打包方式

  • Java Version：JDK版本

  • Version、Name、Description：对应项目的一些描述信息

  • package：自动生成的包，启动类就在这个包下，启动类的名称为Name+Application（如这里的SpringbootCreateApplication）

  • 选择Spring Boot版本，并勾选需要的依赖。

选择的依赖都会在右侧展示。

  • 确认项目名称，选择项目存放路径

  • 查看项目是否完整
